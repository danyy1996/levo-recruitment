# Levo Recruitment

This is the Levo recruitment test bootstrap app

## Building and running on localhost

First install dependencies:

```sh
yarn install
```

To run the webpack dev server:

```sh
yarn start
```

To create a production build:

```sh
yarn build
```

## Testing

To run unit tests:

```sh
yarn test
```
